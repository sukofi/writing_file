# 語尾の連続チェック

買取系SEO記事の初稿で、**同一見出し（h2 または h3）ブロック内**において、**同じ語尾が3回以上連続**していないかを確認し、連続している場合は言い換えて解消する。

## 対象

- ファイル：`01.対策KW/[KWフォルダ]/` 内の **初稿.html**
- 範囲：各 **h2** または **h3** の下にある本文（&lt;p&gt;）の、文末の語尾

## チェックする語尾

主に次のような語尾の**3連続**を避ける。

- **です**（〜です。〜です。〜です。）
- **ます**（〜ます。〜ます。〜ます。）
- **でしょう**（〜でしょう。〜でしょう。〜でしょう。）
- **あります**（〜あります。〜あります。〜あります。）

※「です」「ます」が混在する2連続（例：です→ます）は可。問題になるのは**同一語尾が3回以上続く**場合。

## 手順

1. **見出しごとにブロックを区切る**  
   h2 または h3 ごとに、その見出しから次の同階層の見出しまでの本文を1ブロックとする。

2. **各ブロック内の文末を確認**  
   各 &lt;p&gt; 内の文（。で区切られた単位）の語尾を順に確認する。

3. **3連続を検出したら修正**  
   同じ語尾が3文続いていたら、そのうち1文以上を別の語尾に言い換える。

## 言い換え例

| 連続している語尾 | 言い換え例 |
|------------------|------------|
| 〜です | 〜にすぎません / 〜となります / 〜とされています |
| 〜です | 〜おすすめします / 〜してください（依頼形） |
| 〜ます | 〜でしょう / 〜かもしれません |
| 〜ます | 〜です（です・ますを意図的に混ぜる） |
| 〜でしょう | 〜と考えられます / 〜可能性があります |

## 修正例（実際の文）

**修正前（「です」3連続）**  
「…オークションの取引結果**です**。…なることが一般的**です**。…規約確認が必要**です**。」

**修正後**  
「…取引結果**にすぎません**。…なることが一般的**です**。…利用規約の確認を**おすすめします**。」

## 注意

- 見出し文やリスト項目（&lt;li&gt;）の語尾も、同じブロック内でカウントする。
- 表（&lt;table&gt;）内のセル文は必要に応じてチェック対象に含めてよい。
- 2連続までなら許容し、**3連続以上**になった場合のみ修正する。
