# 薬機法・健康増進法チェック

買取系SEO記事（運動、エクササイズ、美容等）における、**薬機法（医薬品医療機器等法）**、**医療法**、**健康増進法**に基づいた表現の適正化を行う。

## 対象

- ファイル：構成案（`構成：[KW名].md`）、初稿（`初稿：[KW名].html`）
- 対象箇所：h1〜h4見出し、リード文、本文、Q&Aセクション

---

## 1. 禁止・要注意表現とチェックの観点

運動・エクササイズなどのサービスは「医療行為」ではないため、以下の表現は避ける必要がある。

| カテゴリ | NGな表現例 | 理由 |
| :--- | :--- | :--- |
| **治療・完治の暗示** | 治る、完治、解消、根本改善、痛みが消える | 医師法・医療法への抵触。特定の症状が「治る」と断定することはできない。 |
| **身体機能への直接的影響** | 自律神経を整える、代謝を上げる、毒素を出す（デトックス）、免疫力を高める | 健康増進法（誇大広告）への抵触。身体機能の変化を確約する表現は避ける。 |
| **効果の保証・誇大表現** | 激減、必ず、誰でも、効果絶大、100% | 景品表示法・健康増進法への抵触。効果の出方には個人差がある。 |

---

## 2. 推奨される言い換え表現（パラフレーズ）

SEO効果（KW含有）を維持しつつ、安全な表現に言い換える。

| NG表現 | 言い換え案（セーフ表現） |
| :--- | :--- |
| **〜を解消する** | 〜の悩みへのアプローチ、〜をケアする、〜をサポートする |
| **〜が治る** | 〜に寄り添う、〜と向き合う、〜を目指す、コンディショニング |
| **〜を整える** | 〜を維持する、〜のリフレッシュ、〜のリラックス |
| **〜が改善される** | 〜を健やかに保つ、〜をサポートする、〜を目指す |
| **〜が激減する** | 〜がスッキリする、〜が軽くなる、〜を感じやすくなる |

---

## 3. 執筆・チェックの手順

### STEP 1：対象箇所の抽出
構成案または初稿を読み込み、「効果」「悩み」「身体的変化」に言及している箇所（特に出口の見出し）を特定する。

### STEP 2：禁止語句のチェック
「治る」「解消」「整える」「改善」「激減」「解消」などの語句が含まれていないか、またそれらが「断定」や「保証」になっていないかを確認する。

### STEP 3：文脈に合わせた言い換え
「ピラティスは運動である」という前提に基づき、美容・健康維持の範囲内（ケア、サポート、リフレッシュ等）の言葉に置き換える。

### STEP 4：エビデンスの扱い
論文や統計（Cochrane, 厚生労働省等）を引用する場合でも、「〜が証明されているから治る」ではなく、「〜という研究結果が示唆されている」といった中立的な表現に留める。

---

## 4. 修正後の報告形式

修正が必要な箇所がある場合、以下の形式で提案・報告する。

1. **箇所の特定**（見出し名や行数）
2. **現状の表現**（NG箇所を太字に）
3. **懸念点**（どの法律に、なぜ抵触するか）
4. **修正案**（セーフな表現案）
