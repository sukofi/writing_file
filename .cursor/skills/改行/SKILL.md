# 改行ルール（モバイル可読性）

買取系SEO記事の**最終版HTML**で、スマホでも読みやすいように改行と段落分割を行う。参照サンプルはプロジェクト内の **改行.md**。

## 基本方針

- **長い1段落**は、意味の切れ目で**2〜3段落に分割**する。
- 段落と段落のあいだに余白を入れたいときは `<p>&nbsp;</p>` を使う。
- 1文〜2文で1つの `<p>〜</p>` にすると、モバイルで追いやすい。

## 適用箇所

### 冒頭・リード

- 「」のセリフはそれぞれ別 `<p>`。
- 「このように考えていませんか？」のあと → `<p>&nbsp;</p>` → 「この記事では〜」の段落。
- 「この記事では〜」のあと → `<p>&nbsp;</p>` → 「読めば〜」の段落。
- 読者への注意（※買取相場など）の前後で必要に応じて `<p>&nbsp;</p>`。

### 見出しの直後

- **各 h2 の直後**に `<p>&nbsp;</p>` を1行入れてから、最初の本文段落を書く。
- h3 の直後は、本文が短ければ &nbsp; は省略可。長いブロックのときだけ入れる。

### 本文中の段落分割

- **2つ以上の話題**が1段落に入っているときは、話題ごとに分割する。
  - 例：「〜です。<a>〜</a>によれば〜」と「現在流通している一万円札は〜」→ 2段落に分け、あいだに `<p>&nbsp;</p>`。
- **「一方、〜」「そのため、〜」「ただし、〜」** で始まる文は、前の段落と分けて新しい `<p>` にするとよい。
- **手順・リストの前**：「手順は次のとおりです。」は独立した `<p>` にしてよい。その前に「〜手数料はかかりません。」で段落を閉じ、必要なら `<p>&nbsp;</p>`。

### 長い説明の分割

- **3文以上が続く段落**は、文のつながりを見て2段落に分ける。
  - 例：ATM・セルフレジの説明で「メガバンク〜」「一方、コンビニATM〜」「セルフレジは〜」「公表されている情報〜」→ それぞれ短い段落にし、必要に応じて `<p>&nbsp;</p>` を挟む。
- **リスト（post-box-red）の前後**：リストの直前に短い導入文だけの段落、リストの直後に「それぞれ詳しく解説します。」などの1文段落。必要なら &nbsp; で余白を入れる。

### 買取大吉ブロック・ボタン

- **`<div class="btn">`** の前と後に `<p>&nbsp;</p>` を入れる（装飾.md・改行.mdと同様）。
- 店頭・出張・宅配の各 h3 内で、説明文とボタンのあいだに `<p>&nbsp;</p>` を入れる。

### Q&A

- **回答が長い場合**は、「はい/いいえ＋要点」の段落と、「補足・ただし〜」の段落に分ける。あいだに `<p>&nbsp;</p>` を入れる。
  - 例：「はい、使えます。〜無期限で有効です。」→ `<p>&nbsp;</p>` → 「ただし、コンビニATMや〜」

### まとめ

- **まとめの段落**は、次のように分割する。
  - 1段落目：結論（本物の日本銀行券であること・有効な券種）。
  - 2段落目：識別・使えない場合の対処・買取相場の要点。
  - 3段落目：買取大吉の案内。
- 各段落のあいだに `<p>&nbsp;</p>` を入れる。

## 参照ファイル
