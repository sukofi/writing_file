# 最終版装飾

買取系SEO記事の**最終版**（最終版：[KW名].html）に、装飾.mdのルールに沿って装飾を適用する。対象は `01.対策KW/[KWフォルダ]/最終版：[KW名].html`。

## 1. 黄色マーカー（post-text-marker）

- **重要な一文にだけ**付ける。各見出しごとではなく、**記事全体で数箇所（目安：5箇所前後）**に絞る。
- 使い方：`<strong><span class="post-text-marker">〜重要な一文〜</span></strong>`
- 付ける候補の例：
  - 各h2ブロックの「結論・要点」になる1文（D号券の特徴・偽札ではない・使える・買取相場の目安・まとめの核心など）
- **付けてはいけない**：すべてのh3やQ&Aの回答にマーカーを付けること。

## 2. 箇条書きの赤ボックス（post-box-red）

- すべての**ul**および**ol**を `<div class="post-box-red">` で囲む。
- リスト項目は `<b>` で太字にする。
- 「この記事でわかること」は見出しを「＜この記事でわかること＞」とし、中央寄せ：`<p style="text-align: center;"><strong>＜この記事でわかること＞</strong></p>` のうえで同じく `post-box-red` で囲む。

## 3. 内部リンク

- **コラム記事（関連記事）へのリンクのみ** blogcard にする。
  - 形式：`<p>[blogcard url="https://daikichi-kaitori.jp/〜"]</p>`
  - 「【関連記事】」の直後に1行で記載。
- **買取大吉の買取・店舗リンク**（最寄り店舗・店頭/出張/宅配の申し込みなど）は blogcard にせず、**リンクボタン**にする。
  - 形式：`<div class="btn"><b><a href="URL" target="_blank" rel="noopener">表示テキスト</a></b></div>`
  - 例：「最寄りの店舗を確認してみる」「最寄りの『買取大吉』を探してみる」「出張買取の申し込みはこちら」「宅配買取の申し込みはこちら」

## 4. リンクの別タブ

- すべての `<a href="...">` に `target="_blank" rel="noopener"` を付ける。blogcard はショートコードのため、出力側で別タブになる前提でよい。

## 5. 競合サイトのリンク削除（最終入稿時必須）

- **買取メディアなど競合となり得るサイトは、絶対に最終入稿時にリンクを削除すること。**
- 初稿では根拠・参考として外部リンクを入れていても、最終版では競合となる買取系メディア・他社買取サイトへのリンクはすべて削除する。
- 公的機関（日本銀行・国立印刷局・財務省・Wikipedia 等）のリンクは残してよい。競合かどうか判断に迷う場合は削除する。
