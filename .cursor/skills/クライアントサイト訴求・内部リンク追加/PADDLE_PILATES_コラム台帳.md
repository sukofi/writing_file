# PADDLE Pilates コラム台帳

`paddle-gym.com` 向けの内部リンク候補を、毎回Web検索せずに参照できるように整理した台帳。

## 使い方

- クライアントサイトURLが `https://paddle-gym.com/` のときは、まずこのファイルを読む
- 構成の各h2/h3と「話題が続く」記事を優先して選ぶ
- 1つのh2ブロックに最大1〜2本までに抑える
- 足りない場合のみWebSearchやWebFetchで補完し、必要ならこの台帳も更新する

## 参照元

- 公式サイト：https://paddle-gym.com/
- コラム一覧：https://paddle-gym.com/column
- サイトマップ：https://paddle-gym.com/sitemap.xml
- 動的コラムサイトマップ：https://paddle-gym.com/sitemap-dynamic/sitemap-dynamic-column-s--c-slug.xml

## 運用メモ

- まず `sitemap.xml` を見て、`sitemap-dynamic-column-s--c-slug.xml` の有無を確認する
- 新しいコラムが増えていたら、URL・タイトル・主なテーマ・使いやすい文脈を追記する
- 同テーマの記事が複数あるときは、より検索意図に近いものを優先する
- 訴求ブロックでは `motorcontrol` と `experience` を優先候補にしやすい
