# 初稿 Q&A 見出しルール

買取系SEO記事の初稿で**Q&A（よくある質問）**セクションを書く・編集するときの見出し形式。

## 形式

| 階層 | 形式 | 内容 |
|------|------|------|
| **h2** | 【Q&A】見出しタイトル〜〜 | Q&Aブロック全体のタイトル（例：〇〇に関するよくある質問） |
| **h3** | Q.〜〜〜〜 | 質問文そのまま |
| **本文** | &lt;p&gt; | 答え（アンサー） |

各「質問」ごとに **h3 Q.〜** → **本文（&lt;p&gt;）が答え** の順で繰り返す。

## 例

```html
<h2>【Q&A】1964年東京オリンピック記念1000円銀貨に関するよくある質問</h2>

<h3>Q.記念硬貨だけど現金として使える？</h3>
<p>はい、法律上は…（答え）</p>

<h3>Q.1000円銀貨の品位（銀含有率）は？</h3>
<p>昭和39年東京オリンピック記念1000円銀貨幣の品位は…（答え）</p>
```

## 適用タイミング

- 初稿.html の「よくある質問」「Q&A」セクションを新規作成するとき
- 既存のQ&A見出しが上記形式になっていないとき（h2 に【Q&A】がない、h3 Q.の直後に本文が答えでない）に修正するとき
