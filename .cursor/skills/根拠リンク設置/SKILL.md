# 根拠のリンク設置

買取系SEO記事の初稿において、**数値・根拠の提示が必要な箇所**に、ソース（例：ソース.md・ファクトチェック結果）に基づき**アンカーテキストで引用元URL**を設置する。

## 前提

- 対象：`01.対策KW/[KWフォルダ]/` 内の **初稿.html**
- 参照：同フォルダの **ソース.md**（またはファクトチェック・証拠一覧）に記載された引用元URL・検証結果を根拠とする

## 業者サイトのリンク（入稿前に削除するため赤文字で設置）

<span style="color:#c00;">**業者サイト（買取業者・買取店の公式サイト）へのリンクも、根拠として設置してよい。**</span>  
ただし入稿時には競合になるため、あとで削除する。削除しやすくするため、<span style="color:#c00;">**業者サイトへのリンクは必ず赤文字（`style="color:#c00;"`）で設置する。**</span>

## 作業手順

1. **ソースを読む**  
   ソース.md（または同等の文書）で、以下を把握する。
   - 数値・スペックの根拠URL（造幣局・財務省・日本銀行など）
   - 法令・規約の根拠URL（メルカリ規約、財務省FAQなど）
   - 相場・落札データの根拠（Yahoo!オークション、業者サイトなど）
   - 偽物・見分け方などの解説（専門サイト・造幣局・業者サイトなど）
   - **ソースに引用元がない数値・根拠** → **ウェブ検索**で根拠となるURLを探し、**必ず**リンクを設置する。

2. **リンクを付けるべき箇所を特定**  
   初稿の本文で、次のような記述に引用リンクを付ける。
   - 発行枚数・品位・直径・重量などの**数値・スペック**
   - 「日本で初の記念貨幣」「造幣局の公式データ」などの**公式情報の言及**
   - 買取相場・落札価格の**目安や比較**
   - 銀行での換金は額面のみ、記念貨幣は法定通貨、など**制度的な主張**
   - フリマ・規約（出品禁止など）の**ルール**
   - 偽物の有無・見分け方などの**専門的説明**

3. **アンカーで設置**  
   - 根拠となる語句や数値の前後を、**自然なアンカーテキスト**でリンクにする。
   - 外部リンクには `target="_blank"` と `rel="noopener"` を付ける。
   - **業者サイトへのリンク**は、<span style="color:#c00;">`style="color:#c00;"` を付けて赤文字で設置する</span>（入稿前に削除するため）。
   - 例（公式）：`<a href="https://www.mint.go.jp/..." target="_blank" rel="noopener">造幣局の記念貨幣一覧</a>`
   - 例（業者サイト・赤）：`<a href="https://example-kaitori.jp/..." target="_blank" rel="noopener" style="color:#c00;">〇〇買取の解説</a>`

4. **重複を避ける**  
   同じ段落内で同じURLを何度も張りすぎない。必要なら「（参考：〇〇）」のように文末に1回だけまとめてもよい。

## アンカー例

| 内容 | アンカーテキスト例 | リンク先例 |
|------|-------------------|------------|
| 日本初の記念貨幣 | 日本で初めての記念貨幣 | 造幣局 記念貨幣一覧 |
| 品位・スペック | 造幣局の公式データ / 造幣局の「〜」のデータ | 造幣局 該当貨幣ページ |
| 発行枚数 | 造幣局の記念貨幣一覧 / 造幣局のデータ | 造幣局 記念貨幣一覧 |
| 落札相場 | Yahoo!オークション | auctions.yahoo.co.jp |
| 銀行換金は額面のみ | 日本銀行 | boj.or.jp |
| 記念貨幣の出品禁止 | 規約により禁止されています / 規約で禁止 | メルカリ 禁止されている出品物 |
| 法定通貨として使える | 財務省のFAQ「〜」 | 財務省 FAQ該当ページ |
| 偽物・見分け方 | 専門サイトでは偽物の存在や見分け方 / 造幣局の公式仕様 | 専門サイト・造幣局。業者サイトの場合は<span style="color:#c00;">赤文字で設置</span> |
| 買取相場（業者） | 〇〇の買取価格など | <span style="color:#c00;">業者サイトは赤文字で設置。入稿前に削除</span> |

## 注意

- **ソースにない場合はウェブ検索で根拠を探す**：ソース.mdに引用元が書かれていない数値・法令・相場などは、**ウェブ検索**で公式サイト・公的データ・信頼できる一次情報のURLを探し、**必ず**根拠リンクを設置する。検索で見つかった該当ページのURLをアンカーで設置する。
- ソース.mdにURLが明記されていない場合も、上記のとおり検索して採用する。
- 既に「（参考：〇〇）」でリンクがある箇所は、本文中の根拠箇所と重複しない範囲でアンカーを追加する。
- <span style="color:#c00;">**業者サイトへのリンクは赤文字で設置し、入稿前に削除する。**</span>
