# 構成SEOレビュー

SEO記事の構成案を、**「このままで上位表示を狙えるか」**という観点で厳しめに診断する。

## 対象

- `01.対策KW/[KWフォルダ]/構成：[KW名].md`
- 必要に応じて `検索意図：[KW名].md`
- 必要に応じて `*AIO.csv`、`①上位.csv`、`②事前調査.csv`

## 基本方針

- レビューが目的のときは、**先に問題点を指摘**し、要約は最後に短く添える
- 「良い構成です」で流さず、**上位化を阻害する要因**を優先して挙げる
- 構成作成ルールは `構成参照・ルール` を参照する
- ピラティス、ヘルスケア、美容などのテーマでは、必要に応じて `薬機法・健康増進法チェック` も参照する
- ユーザーが修正を依頼していない限り、**レビューのみ**を返す

## 実行手順

1. **対象ファイルを読む**
   - `構成：[KW名].md` を読む
   - あれば `検索意図：[KW名].md` も読む
   - 不足があれば `*AIO.csv` や上位調査系CSVも見る

2. **検索意図と見出し順序を照合する**
   - 冒頭で主検索意図に最短で答えているか
   - `とは/結論 → 理由 → 向いている人 → 期間・頻度 → 比較 → 注意点 → 行動` の流れが自然か
   - 再検索されやすい論点がQ&Aに押し込まれすぎていないか

3. **見出しの網羅性を確認する**
   - 主要論点が抜けていないか
   - 比較、失敗要因、期間、頻度、向き不向き、始め方などが拾えているか
   - h2/h3の粒度がそろっているか
   - h3が単発になっていないか

4. **SERPで勝ちにくい弱点を探す**
   - 競合が共通で扱う論点を落としていないか
   - タイトルや見出しが抽象的すぎないか
   - 「痩せる」のような強いKWで、読者が期待する比較軸や数字に答えられる設計か

5. **E-E-A-T / YMYL観点を確認する**
   - 数字、期間、回数、消費カロリーなどに根拠を置ける構成か
   - 安全性、注意喚起、個人差への配慮があるか
   - 医療・健康効果を断定しやすい危ない見出しになっていないか

6. **商業訴求の自然さを確認する**
   - 情報提供パートより先に売り込みが強く出ていないか
   - 全国系KWに対してローカル訴求が浮いていないか
   - CTA前に十分な中立情報があるか

7. **改善優先度を付けて返す**
   - 致命度が高い順に並べる
   - 必要なら「今すぐ直すならここ」の形で3〜5点に絞る

## 重点チェック項目

### 1. 検索意図一致

- ユーザーの最初の疑問に冒頭で答えているか
- 顕在ニーズだけでなく、不安・比較・失敗回避も拾えているか

### 2. 見出し設計

- 見出しだけで内容が推測できるか
- 抽象語に逃げず、比較軸や条件が明確か
- h2に昇格すべき論点がQ&Aに埋もれていないか

### 3. 根拠設計

- 「2〜3ヶ月」「週2〜3回」などの数字を後で根拠づけられるか
- 一次情報、研究、監修コメントを差し込みやすい構成か

### 4. YMYL安全性

- 痩身、姿勢、痛み、代謝などで断定表現に寄りすぎていないか
- 「治る」「改善する」「必ず痩せる」系の危険な方向に進んでいないか

### 5. 商業性のバランス

- CTAが早すぎないか、強すぎないか
- 店舗訴求がKWの検索意図とズレていないか

## 出力形式

レビューは以下の順で返す。

1. **結論**
   - 「上位表示の可能性はある / 厳しい / 条件付きで狙える」を短く述べる

2. **指摘事項**
   - 致命度が高い順に番号付きで記載
   - 各指摘で以下を含める
     - 何が弱いか
     - なぜSEO上不利か
     - 可能なら該当見出しを引用

3. **良い点**
   - 良い点があれば2〜3点だけ挙げる

4. **優先修正案**
   - 今すぐ直すべき点を3〜5個に絞る

## 評価の目安

- **厳しい**：検索意図ズレ、論点不足、根拠不足、商業色の強さが目立つ
- **条件付きで狙える**：大枠は良いが、根拠設計や比較軸の不足がある
- **上位表示を狙いやすい**：検索意図、網羅性、根拠設計、商業導線のバランスが取れている

## よくある減点ポイント

- 主検索意図への回答が遅い
- 比較ニーズが浅い
- 期間・頻度・回数などの数字に根拠を置けない
- Q&Aに重要論点を逃がしている
- 全国KWなのに特定地域CTAが強すぎる
- YMYL領域なのに安全性・個人差への配慮が薄い
