# 構成作成・参照・ルール基盤

買取系SEO記事の**構成**（構成：[KW名].md）作成時に共通で参照するファイル、見出しルール、出力フォーマット。

## 参照ファイル一覧

| 場所 | 用途 |
|------|------|
| `01.対策KW/[KWフォルダ]/③記事要項.csv` | Who/What/ゴール/How、この記事でしか得られない情報 |
| `01.対策KW/[KWフォルダ]/①上位.csv` | headline 列（競合の h1/h2/h3） |
| `01.対策KW/[KWフォルダ]/②事前調査.csv` | 顕在KW・潜在KW・PAA・共起語 |
| `02.レギュレーション/レギュレーション.md` | 構成・リード・表現・まとめ・画像のルール |
| `02.レギュレーション/構成作成ルール.md` | 見出し階層・結論ベース・KW含有の要約 |

## 見出しのルール

- **階層**：h1 → h2 → h3 → h4 の順。h2 単体可。h3/h4 は同一親の下に2つ以上（単体禁止）。
- **結論ベース**：見出しだけ読んで内容が推測できる。
- **KW**：対策KW・サブKW・LSI を見出しに含める（詰め込みすぎず）。
- **具体表現**：「大きい」→「○cm以上」など数値・固有名詞で書く。
- **MECE＋オリジナル**：競合のトピックを漏れなく含め、差別化用のオリジナル見出しを追加。

## 構成ファイルの出力フォーマット

- **保存先**：`01.対策KW/[対策KWフォルダ]/構成：[KW名].md`
- **見出し表記**：`### h2：[見出し]`、`- **h3**：[見出し]`
- **冒頭**：メインKW・想定読者・順序の考え方を簡潔に記載

## 見出しの標準順序（検索ニーズの流れ）

概要（とは）→ 相場・価格 → 背景（なぜその価格か）→ 高く売れる条件 → 売る際の注意点 → 関連トピック → 売り方・買取方法（CTA）→ よくある質問 → まとめ。
