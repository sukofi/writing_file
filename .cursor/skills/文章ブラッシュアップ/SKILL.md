# 文章ブラッシュアップ

買取系SEO記事の初稿（初稿.html）を対象に、**文章表現の改善・誤字脱字修正・語尾連続チェック**を一括で実施する。

## 対象

- ファイル：`01.対策KW/[KWフォルダ]/初稿：[KW名].html`

---

## 手順

### STEP 1：ファイルを読み込む

初稿HTMLを全文読み込む。

### STEP 2：文章表現ブラッシュアップ

以下の観点で本文（`<p>`タグ内）を精査し、必要箇所を修正する。

| チェック項目 | 修正方針 |
|------------|---------|
| 表現の重複 | 同じ意味の言葉が近くに連続していたら一方を削除または言い換え |
| 不自然な受身形 | 「〜させられる」「〜にされる」など→能動形に変換 |
| 冗長な表現 | 「〜ということが重要となります」→「〜が重要です」など簡潔に |
| あいまいな語尾の重なり | 「〜といえます」「〜望ましいといえます」→どちらか一方を明確な語尾に |
| 唐突な締め文 | まとめ・訴求セクションの末尾が唐突にならないよう文脈に合わせて調整 |
| 読点・接続詞の乱れ | 一文が長すぎる場合は分割、接続詞の重複を解消 |
| 価格・数値の変動注記 | 記事内に具体的な価格・証拠金・利率・相場などの数値を記載している場合は、その数値が「変動するものである」ことを読者に伝える注記（例：「市場の状況により変動します」）を適宜追加する |

> h2リード文・h3本文・まとめ・買取大吉訴求セクションのすべてを対象とする。

### STEP 3：免責文の内容適正化

記事内に `※` で始まる免責文（赤字の注記）がある場合、その内容が記事の実際のトピックと一致しているかを確認し、必要であれば書き換える。

| 記事のトピック | 免責文の方針 |
|--------------|------------|
| 買取相場・査定価格を紹介している | 「記事内で紹介している買取相場などは、買取市場での相場であり、『買取大吉』の買取価格を保証するものではありません。」 |
| 投資・先物・価格チャートなど金融情報を扱っている | 「記事内で紹介している価格・証拠金などは、執筆時点の情報をもとにした目安です。市場の状況により常に変動するため、実際の取引にあたっては必ずご自身で最新情報をご確認ください。」 |
| 市場価格・相場動向の見通しを記載している | 「記事内の価格情報・価格予想はあくまで参考情報であり、将来の価格や利益を保証するものではありません。」 |

> 免責文が上記と内容が異なる場合、記事のトピックに合った文言に変更する。免責文が複数箇所にある場合はすべて確認・修正する。

### STEP 4：誤字脱字チェック

- 明らかな誤字（「的には」の重複、漢字の誤変換など）を修正する
- 記号・数値の全角・半角の乱れを統一する（数字は半角、単位は慣用に従う）

### STEP 5：薬機法・健康増進法チェック

文章ブラッシュアップと同時に、以下の観点で表現の適正化を行う。

| カテゴリ | NGな表現例 | 言い換え案 |
|---------|-----------|-----------|
| 治療・完治の暗示 | 治る、完治、解消、根本改善、痛みが消える | 〜の悩みへのアプローチ、〜をケアする、〜をサポートする |
| 身体機能への直接的影響 | 自律神経を整える、代謝を上げる、免疫力を高める | 〜を維持する、〜のリフレッシュ、〜を目指す |
| 効果の保証・誇大表現 | 激減、必ず、誰でも、効果絶大、100% | 〜が期待できる、〜をサポートする、〜につながりやすい |

> 断定・保証になっていないかを特に注視する。エビデンスを引用する場合も「〜が示唆されている」など中立的な表現に留める。

### STEP 6：語尾連続チェック

**同一h2またはh3ブロック内**で、同じ語尾が**3文以上連続**していないかを確認し、連続していれば言い換えて解消する。

#### チェックする語尾

- **です**（〜です。〜です。〜です。）
- **ます**（〜ます。〜ます。〜ます。）
- **あります**（〜あります。〜あります。〜あります。）
- **なります**（〜なります。〜なります。〜なります。）
- **でしょう**（〜でしょう。〜でしょう。〜でしょう。）

> 「です」→「ます」のように異なる語尾の2連続は許容。**同一語尾が3連続以上**になった場合のみ修正する。

#### 言い換え例

| 連続語尾 | 言い換え |
|---------|---------|
| 〜です | 〜となります / 〜といえます / 〜にすぎません |
| 〜ます | 〜でしょう / 〜かもしれません / 〜です |
| 〜なります | 〜つながります / 〜といえます / 〜果たします |
| 〜でしょう | 〜と考えられます / 〜可能性があります |

---

## 修正後の確認

- 修正箇所をまとめて報告する（箇所・修正前→修正後）
- 誤字脱字・語尾連続の検出件数も明示する
- 薬機法・健康増進法に抵触する表現の修正件数も合わせて報告する
