# 記事編集ワークフロー（対策KW初稿）

対策KW記事の初稿（HTML・MD）を整えるときの手順とルール。レギュレーション（大吉訴求文・構成ルール）に沿って実施する。

---

## 1. 内部リンク（対策KW）の追加

**形式**
- 【関連記事】の下に **KW** と **URL** を2行で記載する。
- **KWはアンカーテキストにしない**。リンクはURLに張る。

```html
<p><strong>【関連記事】</strong><br />
KW<br />
<a href="URL" target="_blank" rel="noopener" style="color:#e00;">URL</a></p>
```

**ルール**
- 内部リンク候補（KW・URL）は別MDやリストで渡される想定。
- **1か所にまとめず、記事の流れ・各見出しの文脈に合わせて分散して配置する。**
- 例：買取相場の説明の直後→「記念硬貨 買取相場」、価値の説明の直後→「オリンピック記念硬貨 価値」、売り方・買取方法の直後→「記念硬貨 買取」。
- リンクはURLを表示し、そのURLを赤字（`style="color:#e00;"`）でリンクにする。KWはリンクにしない。

---

## 2. 語尾の3連続チェック（本文）

**対象**：見出しではなく**本文**（段落の最後の文の語尾）。

**ルール**
- 同一の語尾（例：〜ます、〜です）が**段落で3回以上連続しない**ようにする。
- 見出しをまたぐ連続は可（同じh2/h3ブロック内の連続のみチェック）。

**確認手順**
1. 本文の段落を上から順に並べ、各段落の**最後の文の語尾**を抽出する。
2. 同じ語尾が3つ以上続いている箇所を洗い出す。

**修正**
- 1文だけ語尾を変える（例：〜ます→〜でしょう、〜です→〜おすすめします、〜ます→〜ください）。
- 自然な言い回しを崩さない範囲で変更する。

---

## 3. 買取大吉訴求の重複削除

**ルール**
- 買取大吉の訴求（店頭・出張・宅配の詳細説明＋CTA）は**大吉訴求文.md のブロック1つだけ**にする。
- 記事内の他の「売り方・買取方法」セクションでは、店頭・出張・宅配の**詳細説明とCTAボタンは書かず**、リード文と関連記事リンクのみ残す。
- 本文中の「『買取大吉』で無料査定を…」などの個別訴求文も削除する。

**構成**
- 大吉訴求ブロックは **Q&A の前** に配置する。
- 最終的な並び：**売り方・買取方法（リード＋関連記事） → 大吉訴求 → Q&A → まとめ**

---

## 4. HTML と MD の同期

- 語尾の修正や訴求の削除・移動は**初稿.md と 初稿.html の両方**に反映する。
- MD を先に直し、その内容で HTML を更新するか、同時に同じ変更を両ファイルに加える。

---

## 5. Googleドキュメント用の出力

- 貼り付け用には **初稿.md** を用意する（見出しは # / ## / ###、リストは -、表はパイプ表）。
- 見出しサイズなどを反映させたい場合は、**HTMLをGoogleドライブにアップロードし「Googleドキュメントで開く」**と、見出し・表・リストが保持されやすい。
